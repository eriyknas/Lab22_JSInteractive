## Лабораторная работа №22. Интерактивные веб-приложения на JavaScript
## Описание проекта
Калькулятор, разработанный в рамках лабораторной работы. Он поддерживает базовые математические операции, имеет валидацию ввода и обрабатывает ошибки.
## Функциональность
### Поддерживаемые операции
- Сложение
- Вычитание
- Умножение
- Деление
### Ограничения
- Максимальная длина выражения не ограничена
- Поддерживаются целые числа и десятичные дроби
- Не поддерживаются скобки и сложные математические функции
### Обработка ошибок
- Деление на ноль - выводится сообщение "Ошибка"
- Пустое выражение - при нажатии `=` ничего не происходит
## Архитектура
### Структура файлов
- index.html - HTML-разметка калькулятора
- style.css - стили для калькулятора
- calculator.js - класс Calculator
- main.js - инициализация
- img - папка со скриншотами
### Описание класса Calculator
- `constructor(displayElement)` - Инициализация свойств: display, expression, hasError
- `init(buttons)` - Навешивание обработчиков событий на кнопки
- `handleButtonClick(value)` - Основной обработчик нажатий кнопок
- `calculate()` - Вычисление математического выражения
- `safeEvaluate(expr)` - Безопасное вычисление с валидацией (замена eval)
- `isValidExpression(expr)` - Проверка корректности выражения
- `clear()` - Очистка калькулятора
- `updateDisplay()` - Обновление поля ввода
- `showError()` - Отображение ошибки
- `isOperator(char)` - Проверка, является ли символ оператором
- `isLastCharOperator()` - Проверка последнего символа на оператор
- `hasDuplicateDot(expr)` - Проверка на несколько точек в числе
- `validateInput(value)` - Валидация вводимых символов
## Логика работы
1. Пользователь нажимает кнопку на калькуляторе
2. Если калькулятор в состоянии ошибки и нажата не кнопка "C", происходит автоматическая очистка
3. **Обработка специальных кнопок**:
   - `C` → вызов `clear()` (сброс выражения и ошибки)
   - `=` → вызов `calculate()` (вычисление результата)
5. Обновление выражения и отображение на дисплее через `updateDisplay()`
### Валидация ввода
```javascript
// Для точки (.)
- Запрещает ввод второй точки в текущем числе
- При пустом выражении или после оператора автоматически добавляет "0"

// Для оператора (+ - * /)
- Запрещает оператор в начале выражения (кроме минуса)
- При повторном вводе оператора заменяет предыдущий

// Для цифр
- Всегда разрешены
```
### Процесс вычисления
- Проверка на пустое выражение
- Удаление оператора в конце
- Обновление состояния — при успешном вычислении
- Обработка ошибки — при неудаче
